## Quick Commands

### Run the simulation (default: 30s × 3 tests)
```bash
cd tests
python3 ai_human_network_simulation.py
```

### Custom configuration
```bash
# From tests directory
cd tests

# 60 seconds per test, 5 iterations
python3 ai_human_network_simulation.py --duration 60 --tests 5

# Reproducible results with seed
python3 ai_human_network_simulation.py --seed 42

# Enable/disable scorer
python3 ai_human_network_simulation.py --scorer
python3 ai_human_network_simulation.py --no-scorer
```

## What This Tests

The simulation models real-world AI-to-human communication patterns:

### Message Types
- **Human Queries**: Short questions, medium context-rich queries, long detailed questions
- **AI Responses**: Brief confirmations, step-by-step explanations, comprehensive answers with code
- **Data Formats**: Plain text, JSON structures, code snippets, error messages

### Network Characteristics
- **Realistic Timing**: 10-100ms intervals between messages
- **Variable Size**: 19-600 character messages
- **Conversation Flow**: Human query → AI response pairs
- **Method Diversity**: Automatic selection of optimal compression method

### Metrics Collected
- **Compression**: Ratios, methods used, consistency
- **Performance**: Latency, throughput, processing time
- **Reliability**: Error rates, message success
- **Consistency**: Variance across test runs
